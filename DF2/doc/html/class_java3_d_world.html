virtual void Java3DWorld::translate ( int  dx,
int  dy,
int  d 
) [virtual]

この空間を移動する。 dx, dy に (Java3D#ANGLE_MAX / 4096) に掛けた値が Java3D#va, Java3D#vb に加えられる。 Java3D#va, Java3D#vb が Java3D#ma, Java3D#mb に加えられる。 16 を Java3D#ma, Java3D#mb で回転した値が Java3D#vx, Java3D#vy, Java3D#vx になる。 Java3D#vx, Java3D#vy, Java3D#vx を d で掛けた値が Java3D#mx, Java3D#my, Java3D#mx に加えられる。

引数:
dx x 方向の加速度。
dy y 方向の加速度。
d 移動量。
