int Java3D::render_mode

このオブジェクトの描画方法のフラグ。

以下の描画フラグのいずれかを指定する。
RENDER_POLYLINE : ワイヤーフレームを描画。
RENDER_POLYFRAME : ポリゴンの枠を描画。
RENDER_POLYGON : ポリゴンを描画。
RENDER_STRING : 文字列を描画。

以下のシェーディングフラグを1つ以上指定できる。
SHADING_GOURAUD : 各頂点の色で変化させる。
SHADING_ALPHA : 透過。

以下のライティングフラグを1つ以上指定できる。
LIGHTING_DIFFUSE : ライティング。
LIGHTING_FOG : 遠くに行くほど暗くなる。

以下のポリゴンバッファへの追加方法のフラグのいずれかを指定する。
APPEND_NONE : ポリゴンバッファへの追加方法。
APPEND_FRONT : ポリゴンバッファへの追加方法。
APPEND_BACK : ポリゴンバッファへの追加方法。
APPEND_SORT : ポリゴンバッファへの追加方法。
