virtual void Java3DModel::transform (  )  [virtual]

このモデルを座標変換・投影する。

モデルの座標変換: parent が null である場合は、モデルの座標を絶対座標として、座標変換を行わない。 parent が null でない場合は、parent からの相対座標・角度を 絶対座標・角度として座標変換を行う。 変換後の座標の情報は trxxxx に設定される。

各頂点の座標変換・投影:
