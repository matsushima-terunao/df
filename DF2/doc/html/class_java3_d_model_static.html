static int Java3DModelStatic::get_brightness_color ( int  color,
int  brightness 
) [static]

指定された明るさで色を正規化。

引数:
color 正規化する元の色(RGB)。
brightness 明るさ。
戻り値:
正規化された色。 
              brightness &lt= 0: 0xff000000
              brightness &gt= BRIGHTNESS_MAX - 1: 0xffffffe
              brightness &lt= BRIGHTNESS_BORDER: color * brightness / BRIGHTNESS_BORDER
              brightness &gt BRIGHTNESS_BORDER:
                  color + (256 - color) * (brightness / BRIGHTNESS_BORDER) ^ 2
                                        / (BRIGHTNESS_MAX / BRIGHTNESS_BORDER) ^ 2
